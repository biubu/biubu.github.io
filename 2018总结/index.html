2018年总结

杂文2018总结
  • 2018总结
  • 杂文

这篇文章是写给自己的,盘点一下自己的这一年。

吃着泡面回顾这一年，五味杂陈，这一年想明白了很多事，生活过的一塌糊涂，怎料想会落到如此地步。工作还好吧，学了很多新东西，这一年多业余时间宅完了，越来越讨厌无效的社交，也越来越恐惧社交，只喜欢和熟悉的老友偶尔叙叙旧，聊聊天，仅此而已。这一年，看了几本书，对自己的影响比较大。如果用一个词语来形容，那我觉得孤独最合适，并且自己享受这种孤独，只不过，享受孤独是从开始卸载游戏开始的，用以前来打游戏的时间看看书，看看新闻，看看资料，挺充实的，是充实的感觉，让我享受孤独。

技术方面

  1. 2018总结：

    我爱我的工作，我爱我的前端开发，我爱我想在所学的一切，打心底里爱。没有什么比看着自己写的项目上线运行更爽的事情了，如果用一个关键词来总结，应该是折腾。

    如果回归到这一年，那应该是迷茫而充实。因为危机感产生迷茫，危机感源于自己技术的面不够广，所以迷茫的想一下子多学几种语言，当然选择了Python，学了几个月，没有实际的项目来实践，学完了，也忘完了，但是不后悔，如果哪天用起来，再去学的话会好点吧。至少，对比着 Python和
    JavaScript，自己还是有点收获的。同样的，学了nodejs，和前面的一样，了解了个皮毛，用 vue+ssr + express + mysql写了个小服务在线上跑着,并且还支持自定义查询,然后呢,用 nginx转发出来,可以通过域名访问。

    这一年折腾了好多东西，首先是操作系统换了三个，哈哈哈，从 Windows 10 换到 Ubuntu,从 Ubuntu 换到 Deepin,从 Deepin 换到 MacOS，现在就是用
    MacOS开发，爽的不要不要的。不折腾，留着那些闲时间来干啥，反正电脑也折腾不坏，有啥可怕的。

    正是因为折腾，才对 Linux系统了解了一点，所以才会折腾 nginx，才会自己搭建 nodejs 服务。我不怕折腾，不断的折腾是一个探索学习的过程，一点一点的积累，总会在某一天有用的。

  2. 2019计划:：

    如果技术方面给2019年定个计划，对比 2018
    的迷茫，我会更专注于前端，世界上那么多的编程语言，我是学不完的，并且，自己的前端水平还有提升空间，专注于前端，做一个专业的前端。虽然技多不压身，同时，贪多嚼不烂。在同样的时间里，要深度，就得牺牲广度。力争在2019年，多学习主流框架的源码，主流的用的最熟的就是 vue了，那就从 vue 源码开始吧。

生活方面:

  1. 2018总结:

    虽然在2018年最后的几个月里,一直强调好好的爱自己,但实际情况自己清楚就行了。好也罢，坏也罢，坦然接受，好的继续保持，不好的继续改进。

    这一年,自己过的波澜不惊,平平淡淡,只能说,亏欠家人的太多了,
    亏欠自己太多了。庆幸的是，一切在向好的方向发展。不想在漂了，漂来漂去，都快把自己的灵魂搞丢了。父母在哪，家就在哪里，趁还来得及，多陪伴他们才是真的，自己从来没做过后悔的事，不要等来不及了，再去后悔，这可不是我的风格。

  2. 2019计划:

    2019，学习着爱上生活，如果困难，就要多逛逛菜市场，菜市场到处都是生机。现在终于明白，当时母亲逼着学做饭是为什么了，如果自己不会做饭，只能去外面吃，天天去外面吃，那能叫生活吗？慢慢的尝试着，学习着做两个简单的小菜；慢慢的尝试着，找回一点点家的感觉；慢慢的尝试着，学会承担家庭的责任。毕竟，年龄摆在那里了，何必跟年龄过不去呢。

工作方面

  1. 2018总结:

    工作方面还好吧,第二季度获得了优秀员工,老板亲自发了个红包,记得当时的获奖感言是从成本的角度发言的。只记得一点，就是想办法核算自己每行代码值多少钱。哈哈哈，结果到年底一看，我才是整个年度提交代码天数最多的。有图为证:

    这么算下来,我的每行代码才是最不值钱的😂😂😂。平心而论，今年的工作尽职尽责了，推进代码规范的实施，研究 ArcGIS API for JavaScript 并输出文档，维护项目模板,团队技术分享……正常的开发任务之外，尽可能的想办法提升工作效率，对我，对公司，都问心无愧。有提升的地方，就是对待新同事的语气方面吧，这方面得多注意一下，谁都是从新人过来的，但是我不能仗着自己会的比他们多就对他们态度不好，这一点得持续改进，争取心平气和的和他们沟通。

  2. 2019计划

    新的工作,新的城市,新的环境,挑战与机遇并存,加油吧！

    保持初心不变，好好的工作，好好的爱自己的工作。
